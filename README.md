# Object Lessons — The Recurring Objects of Cinema

An editorial website dedicated to exploring the taxonomy and significance of recurring objects in film. 

## Concept
This project serves as an ORGAN-V-adjacent editorial surface, categorizing cinematic objects and analyzing their narrative, symbolic, and visual weight across different eras and genres. 

## Development
Built with [Astro](https://astro.build).

```sh
npm install
npm run dev
```
